Fundamentos de JavaScript

Anotaciones generales

  • Nunca utilizar la palabra clave var para declarar variables. Utilizar siempre la palabra clave let ya que ésta respeta el ámbito donde se declara la variable

  • Las constantes tienen que ser declaradas e inicializadas en la misma línea de código. Las variables sí pueden ser declaradas y ser inicializadas después. Las constantes trabajan de manera diferente dependiendo del tipo de dato que se utilice (primitivo o compuesto)

Funciones

Existes varios tipos de funciones:

  • Funciones declaradas:

    Las funciones declaradas se pueden invocar en cualquier parte del código, inclusive antes de haberla declarado Las funciones declaradas son las que tienen antepuesta la palabra function

  • Funciones anónimas/expresada

    Las funciones anónimas/expresada son aquellas que se le asignan como valor a una variable o constante.
    Estas también llevan la palabra function, pero no pueden ser invocadas antes de ser declaradas.

  • Funciones flecha: (=>)

Arreglos en JS

Los arreglos en JS son colecciones de elementos en una variable o constante.
Estos pueden guardar distintos tipos de datos en cada posición. No olvidar que el índice de los arreglos comienza con el cero (0)
